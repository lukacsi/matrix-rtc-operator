# MRO-024: Synapse service + HTTPRoute

## Description
Create the networking resources for Synapse. synapse-service: ClusterIP service on port 8008. synapse-httproute: HTTPRoute for matrix.{domain} routing to the synapse service, with parentRefs sourced from the networking view.

## Acceptance Criteria
- [ ] ClusterIP Service created on port 8008
- [ ] HTTPRoute created for matrix.{domain}
- [ ] parentRefs populated from MatrixNetworkingView
- [ ] curl matrix.{domain}/_matrix/client/versions returns 200

## Notes
The HTTPRoute parentRefs come from the networking view's gateway refs, allowing flexible Gateway API integration.
