# MRO-202: Matrix federation support

## Description
Enable Matrix federation for inter-server communication. Port 8448 routing, SRV DNS records, and federation tester integration.

## Acceptance Criteria
- [ ] Port 8448 exposed and routed for federation
- [ ] SRV DNS records documented/automated
- [ ] Federation tester passes for the domain
- [ ] Users on federated servers can communicate

## Notes
Federation requires careful DNS and TLS configuration. The .well-known approach may simplify SRV record requirements.
