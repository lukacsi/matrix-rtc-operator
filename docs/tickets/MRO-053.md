# MRO-053: ServiceMonitor templates

## Description
Create ServiceMonitor resources for Prometheus scraping. Synapse: /_synapse/metrics endpoint. LiveKit: /metrics endpoint. Gated by metrics.serviceMonitor.enabled flag in the CR spec.

## Acceptance Criteria
- [ ] ServiceMonitor for Synapse targeting /_synapse/metrics
- [ ] ServiceMonitor for LiveKit targeting /metrics
- [ ] ServiceMonitors gated by metrics.serviceMonitor.enabled
- [ ] Prometheus discovers and scrapes metrics when enabled
- [ ] No ServiceMonitors created when disabled

## Notes
Requires prometheus-operator CRDs to be installed in the cluster.
