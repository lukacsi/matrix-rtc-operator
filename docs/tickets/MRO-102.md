# MRO-102: FR-4: Auto-generate ValidatingWebhookConfiguration

## Description
dcontroller should auto-generate ValidatingWebhookConfiguration from CRD OpenAPI schema. This rejects invalid specs at admission time, before they reach the pipeline, providing better user feedback.

## Acceptance Criteria
- [ ] Issue filed on dcontroller repo
- [ ] Proposed generation approach described
- [ ] Example showing invalid CR rejection included

## Notes
Currently invalid CRs are silently accepted and may produce unexpected pipeline outputs. Admission webhooks provide immediate feedback.
