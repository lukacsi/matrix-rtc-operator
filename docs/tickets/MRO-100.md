# MRO-100: FR-2: Auto-generate RBAC from Operator CR

## Description
dcontroller should auto-generate ClusterRole rules from declared sources/targets in Operator CRs. Enterprise clusters won't grant blanket access — operators need least-privilege RBAC that matches exactly what their pipelines read and write.

## Acceptance Criteria
- [ ] Issue filed on dcontroller repo
- [ ] Clear use case from matrix-rtc-operator included
- [ ] Proposed API for RBAC generation described

## Notes
This is a significant adoption barrier for enterprise environments where security teams review RBAC grants.
