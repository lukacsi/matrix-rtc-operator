# MRO-054: PodDisruptionBudget templates

## Description
Create PodDisruptionBudget resources for all components. Configurable minAvailable/maxUnavailable per component. Gated by pdb.enabled flag in the CR spec.

## Acceptance Criteria
- [ ] PDB created for each component when pdb.enabled=true
- [ ] PDB values configurable per component
- [ ] No PDBs created when pdb.enabled=false
- [ ] kubectl drain respects PDBs when enabled

## Notes
CNPG has its own PDB management (enablePDB field on Cluster CR) -- do not duplicate for the database component.
