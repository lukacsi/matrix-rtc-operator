# MRO-104: FR-7: dcontroller-common Helm library chart

## Description
Propose a bitnami/common-style Helm library chart for dcontroller operators. Standardized templates for naming conventions, labels, RBAC, security contexts, and chart skeleton. Reduces boilerplate across dcontroller-based operators.

## Acceptance Criteria

- [ ] Comparison with bitnami/common approach

## Notes
