# MRO-040: lk-jwt-service deployment pipeline

## Description
Deploy the lk-jwt-service that bridges Matrix authentication with LiveKit. Deployment with env vars: LIVEKIT_URL (from LiveKitStack), LIVEKIT_KEY/SECRET (from LiveKit Secret via DecodeSecretData), LIVEKIT_FULL_ACCESS_HOMESERVERS (from MatrixStack domain). Service on ClusterIP port 8080. HTTPRoute for matrixrtc.{domain}/livekit/jwt.

## Acceptance Criteria
- [ ] lk-jwt-service Deployment created with correct env vars
- [ ] LIVEKIT_URL sourced from LiveKitStack
- [ ] LIVEKIT_KEY/SECRET sourced via DecodeSecretData
- [ ] LIVEKIT_FULL_ACCESS_HOMESERVERS set to MatrixStack domain
- [ ] ClusterIP Service on port 8080
- [ ] HTTPRoute for matrixrtc.{domain}/livekit/jwt
- [ ] POST to matrixrtc.{domain}/livekit/jwt with Matrix credentials returns LiveKit JWT

## Notes
This is the critical bridge component enabling Matrix-authenticated users to join LiveKit rooms.
