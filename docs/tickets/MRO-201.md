# MRO-201: CNPG backup integration (ScheduledBackup)

## Description
MatrixStack creates CNPG ScheduledBackup CRs when database.mode=managed and backups are enabled. Support Barman/S3 destination configuration.

## Acceptance Criteria
- [ ] MatrixStack spec supports backup configuration (schedule, destination)
- [ ] ScheduledBackup CR created when backups enabled
- [ ] Barman/S3 destination configurable
- [ ] Backups run on schedule
- [ ] No ScheduledBackup when backups disabled or mode=external

## Notes
Requires S3-compatible storage (MinIO or cloud S3) for backup destination.
