# MRO-064: Homelab deployment via ArgoCD

## Description
Deploy the full matrix-rtc-operator stack on the homelab workload cluster via ArgoCD. Configure ArgoCD apps, SOPS-encrypted secrets, DNS records, and STUNner port forwarding.

## Acceptance Criteria
- [ ] ArgoCD Application CRs created for matrix-rtc-operator
- [ ] SOPS secrets configured for Matrix credentials
- [ ] DNS records created (matrix.lukacsi.org, element.lukacsi.org, etc.)
- [ ] STUNner port forwarding configured for WebRTC
- [ ] Full stack running on homelab workload cluster
- [ ] End-to-end call test passes on homelab

## Notes
Follow the existing homelab ArgoCD app pattern: Helm app + single -manifests app for CNPG, HTTPRoute, PVCs, backup cronjobs.
