# MRO-203: Standalone LiveKit chart extraction

## Description
Extract LiveKitStack as a separate, standalone Helm chart that can be used independently of the Matrix components. Share common templates via the dcontroller-common library chart.

## Acceptance Criteria
- [ ] LiveKitStack Helm chart extracted to separate repository/directory
- [ ] Chart works standalone without Matrix CRDs
- [ ] Shared templates via dcontroller-common library
- [ ] matrix-rtc-operator can use the extracted chart as a dependency

## Notes
This enables LiveKit-only deployments and promotes reusability of the dcontroller patterns.
